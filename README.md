# Smart Store Manager 📦

Ein Java-basiertes Inventarsystem zur Analyse und Verwaltung von Lagerbeständen. Dieses Projekt wurde entwickelt, um reale Herausforderungen im Einzelhandel (inspiriert durch meine Erfahrung bei Tedi) durch Automatisierung zu lösen.

## 🚀 Features
- **Lagerbestands-Analyse:** Automatische Identifizierung von Produkten mit kritischem Bestand (Nachbestellungs-Logik).
- **Finanz-Check:** Berechnung des Gewinns pro Einheit und des Gesamtwerts des Inventars.
- **Effizienz:** Nutzung von **Java Stream API** für eine performante Datenverarbeitung.
- **Architektur:** Trennung von Datenmodell (Model) und Geschäftslogik (Service).

## 🛠 Technologien
- **Sprache:** Java (JDK 25)
- **Build-Tool:** Maven
- **Konzepte:** Objektorientierte Programmierung (OOP), Encapsulation, Streams, Git.

## 📖 Hintergrund
Während meiner Tätigkeit im Einzelhandel habe ich gemerkt, wie zeitaufwendig die manuelle Bestandsprüfung ist. Dieses Tool zeigt, wie man durch Software-Engineering die Effizienz im Tagesgeschäft steigern kann.
